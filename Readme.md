# United Arts: https://github.com/ecumenic/united-arts
![Our Logo](https://github.com/ecumenic/united-arts/blob/master/United%20Arts.png)

1. **Project:** Following the Ecumenic Creative Operations (https://ecumenic.github.io) methodology (https://github.com/ecumenic/project) this project, in compliance to the United Nations Business Handbook for Partnerships (https://bit.ly/un-business-partnerships), and UNESCO Private Engagement Objectives (https://bit.ly/unesco-engagement) and Strategies (https://bit.ly/unesco-partnerships), develops an ecomuseologic sustainable ecocuratorial scheduled seasonal exhibit.

2. **Process:** We gathered from the original file a websites links database (database.txt), used Wget (https://www.gnu.org/software/wget) to bulk download it (wget -i database.txt -t3) in linux console.
